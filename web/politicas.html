Políticas y Privacidad
Uso de datos personales

Capibara Tours utiliza los datos del cliente únicamente para registrar reservas, confirmar pagos, coordinar salidas turísticas y mantener comunicación relacionada al servicio contratado.

Datos solicitados

El sistema puede solicitar nombres, apellidos, documento, teléfono y correo electrónico. Estos datos permiten identificar correctamente al cliente y evitar duplicidad de registros.

Reservas y pagos

La reserva queda registrada con el paquete elegido, cantidad de personas, fecha de viaje y pago correspondiente. Cualquier cambio debe realizarse antes de la fecha programada.

Seguridad

El acceso al sistema se realiza mediante usuario y contraseña. Las áreas administrativas están protegidas para evitar accesos no autorizados.
